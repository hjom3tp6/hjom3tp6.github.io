Posted on 2021-07-13 Edited on 2021-07-18 In 程式 Disqus:
紀錄關於用hexo + github page 架blog

關於用hexo + github page 架blog，google文章實在太多，因此僅記錄此次架設最有幫助的知識點來源:

參考文章

  • IT 邦幫忙鐵人賽-試著學 Hexo，這系列文實在太詳盡，但是因為我已經用到Hexo 5了，所以少部分還是要到Hexo文件去查
  • Theme選用Next。另外因為選擇github action自動部屬，這時如果用git clone主題，push整包專案要用git submodule進行管理。我選擇使用 npm install hexo-theme-next 的方式安裝Theme比較單純一些。
  • 留言使用Disqus，方法在此。
  • Godaddy 買域名、DNS用Cloudflare，設定DNS坑比較多，請直接參考Github page自訂域名。
  • Hexo專案push上去後，發現github page 自訂域名跑掉，找到原因後，解決方法是在hexo source目錄底下新增一個 CNAME 檔案。參考文章。
